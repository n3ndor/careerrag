nagysolution.com is Nandor's portfolio site and the hub for his public projects.
Built with Next.js, TypeScript, and Tailwind CSS, deployed on Vercel.

It links to the live portfolio projects, each running on its own subdomain:

- https://jobradar.nagysolution.com (tech-job market intelligence)
- https://dopaminebuy.nagysolution.com (fake shopping game PWA)

The site is performance-tuned (static-first rendering, deferred background media,
minimal JavaScript on first paint) and hosts the CareerRAG chat assistant.
